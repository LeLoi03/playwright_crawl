 EIE2025 Call for Abstracts  

 The EIE2025 abstract submission portal is open!  
 You can now submit your abstract for the 2025 European Implementation Event.  
 The submission portal will close on January 12, 2025   . You can submit abstracts in various formats, described briefly below and in greater detail here  . Please have a good look at all formats before you start your submission process.  
 Please email  us if you have any questions about the abstract submission process.  

 Submit an abstract to the EIE2025!     

 Ride the Knowledge Wave (oral presentation)  
 In a Ride the Knowledge Wave  session, event attendees learn about and discuss the latest implementation research findings.  

 Walk the Talk (symposium)  
 A Walk the Talk  session allows for shared presentations, debate, idea development, and problem-solving.  

 The Open Implementation Fishbowl (debate)  
 An Open Implementation Fishbowl  session is for lively and open debate on hot topics in implementation.  

 P 3  - Promise, progress, problems (poster)   
 In a P3—Promise, Progress, Problems  session, event attendees learn about promising implementation work that may be in progress or has already generated interesting findings.   

 The EIE2025 Implementation Slam  
 The EIE2025 Implementation Slam  is for those passionate about creative, entertaining implementation knowledge dissemination.  
 An example of a slam can be found here  (Ossama Khalaf at the #56 Science Slam Berlin).  

 Early Career Implementation Professional (ECIP) Keynote  
 The ECIP keynote  provides early career implementation professionals with an opportunity to present their work in front of a large audience.
